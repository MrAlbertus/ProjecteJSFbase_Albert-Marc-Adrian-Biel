# Projecte ReBit - Desenvolupament Web i Firebase

Aplicació web per localitzar punts de reciclatge sota el principis bàsics de Green IT.

## Característiques
* **Tecnologies**: JavaScript, HTML, CSS i Firebase.
* **Persistència**: Ús de Firestore per a la gestió de dades NoSQL.
* **Sostenibilitat**: Disseny amb Mode Fosc i optimització de peticions.
* **Mapa**: API de Leaflet per a la visualització d'ubicacions.

## Estructura de dades (Col·lecció dels punts)
* **name**: Nom del recurs (String).
* **location**: Coordenades (GeoPoint).
* **type**: Llista de residus (Array).
* **Parking**: Disponibilitat (Boolean).
* **Schedules**: Horaris (Map).
